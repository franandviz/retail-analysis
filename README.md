### About the Project
This analysis looks at the buying behaviour of customers through EDA (Exploratory Data Analysis) as well as Market Basket Analysis to refine the findings and offer recommendations. The analysis uses a Customer Shopping Behaviour Dataset from Kaggle, created by Ayesha Siddiqa.

### Objective
To pull out insights about customer behaviour and use Market Basket Analysis to recommend actions this shop could make to increase sales from both a unit and revenue point of view.
